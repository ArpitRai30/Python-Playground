## 🚀 Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package installer)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/ArpitRai30/Python-Playground.git
cd Python-Playground
```

2. (Optional) Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## 🎓 Examples by Category

### Basics (Start Here!)
- **Variables**: Learn about variable assignment, naming conventions, and scope
- **Data Types**: Numbers, strings, lists, dictionaries, sets, and tuples
- **Operators**: Arithmetic, comparison, logical, and bitwise operators
- **Control Flow**: If/else statements, for/while loops, and loop control
- **Functions**: Function definition, parameters, return values, and lambda functions

### Intermediate
- **Object-Oriented Programming**: Classes, inheritance, encapsulation, and polymorphism
- **Data Structures**: Stacks, queues, linked lists, trees, and graphs
- **Algorithms**: Sorting, searching, dynamic programming, and complexity analysis
- **File Handling**: Reading/writing files, CSV, JSON, and binary operations
- **Error Handling**: Try/except blocks, custom exceptions, and debugging

### Advanced
- **Web Development**: Flask applications, REST APIs, and web forms
- **Database**: SQLite operations, SQL queries, and database design
- **API & Web Scraping**: HTTP requests, API consumption, and web scraping
- **Testing**: Unit tests, pytest, mocking, and test-driven development

### Projects
- **Beginner**: Number guessing game, calculator, to-do list, password generator
- **Intermediate**: Web scrapers, API clients, file organizers
- **Advanced**: Full web applications, data analysis tools, automation scripts

## 🏃‍♂️ Running the Examples

Each directory contains standalone Python files that can be run independently:

```bash
# Run basic examples
python basics/variables/variable_examples.py
python basics/data_types/string_types.py

# Run intermediate examples
python intermediate/oop/class_examples.py
python intermediate/algorithms/algorithm_examples.py

# Run advanced examples
python advanced/database/sqlite_examples.py
python advanced/web_development/flask_basics.py

# Run projects
python projects/beginner/beginner_projects.py
python utils/utility_scripts.py
```

## 📖 Learning Path

### For Beginners
1. Start with `basics/variables/`
2. Move through `basics/data_types/`
3. Learn `basics/operators/` and `basics/control_flow/`
4. Master `basics/functions/`
5. Try projects in `projects/beginner/`

### For Intermediate Learners
1. Explore `intermediate/oop/`
2. Study `intermediate/data_structures/` and `intermediate/algorithms/`
3. Practice with `intermediate/file_handling/` and `intermediate/error_handling/`
4. Build projects in `projects/intermediate/`

### For Advanced Users
1. Dive into `advanced/web_development/`
2. Learn `advanced/database/` operations
3. Master `advanced/api_and_web_scraping/`
4. Implement `advanced/testing/` practices
5. Create projects in `projects/advanced/`
