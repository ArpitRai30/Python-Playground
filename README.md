# 🐍 Python-Playground

Welcome to **Python-Playground** - A comprehensive collection of Python projects and implementations by Arpit Rai.

## 📁 Repository Structure

### 🏢 Employee-Management-System
A complete **Employee Management System** built with Python using Object-Oriented Programming principles.

**Features:**
- Manage Full-Time, Part-Time, and Manager employees
- Salary calculations and payroll reports
- JSON-based data persistence
- Interactive console interface

**Technologies:** Python 3, OOP, Abstract Base Classes, JSON

📂 **Files:**
- `emp_management.py` - Main application
- `README.md` - Project documentation

### 🤖 Machine-Learning

#### PRODIGY_ML_01 - House Price Prediction
Linear Regression model to predict house prices based on key features.

**Features:**
- Predict house prices using square footage, bedrooms, and bathrooms
- Model evaluation with MSE and R² score
- Data visualization with matplotlib

**Technologies:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn

📂 **Files:**
- `ML1.py` - House price prediction model
- `README.md` - Project documentation

#### PRODIGY_ML_02 - Customer Segmentation
K-Means clustering for customer segmentation analysis.

**Features:**
- Customer segmentation using demographic and spending data
- Elbow method for optimal cluster selection
- Interactive visualizations of customer segments

**Technologies:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn

📂 **Files:**
- `ml2.py` - Customer segmentation model
- `README.md` - Project documentation
